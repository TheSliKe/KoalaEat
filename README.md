# KoalaEat

## Prérequis

* PHP 8.0
* Mysql 8.0
* Symfony 6.0
 
## Installation

Récupérer le projet :

```bash
git clone https://github.com/TheSliKe/KoalaEat.git
cd KoalaEat
```

Faire un composer install : 

```bash
composer install
```
## Lancer l'application

À la racine du projet :

 ```bash
symfony server:start
```

Inscription sur :
```bash
localhost:8000/register
```

Connexion sur :

```bash
localhost:8000/login
```
